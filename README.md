# Oxify Labs

Where systems architecture meets long-term compounding.

## Overview
Oxify Labs is an engineering consultancy focused on building resilient software systems that embrace complexity and deliver compounding returns. We prioritize stability, maintainability, and architectural correctness.

## Stack
- **Framework**: [Next.js](https://nextjs.org/)
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Visualization**: React Three Fiber (R3F)

## Development

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Philosophy
- **Narrative Sharpness**: Clear, declarative communication.
- **Authority**: Showing thinking through artifacts, not just marketing.
- **Restraint**: Design that ages well.
